# Abstract - Kali Linux Revealed


## What is that course?
+ Kali Linux is comprised of many powerful tools but you cannot put them to good use if you don't master the underlying operating system. 
+ This course covers everything you need to know to be able to effectively use and deploy Kali Linux.
+ This course will discuss basic Linux usage for beginners, Debian package management and usage, Kali installation, configuration, security, and advanced Kali usage including how Kali fits within the enterprise and Kali's role in various phases of a security assessment.
