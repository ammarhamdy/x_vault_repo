## Recent Trends:
+ Bring Your Own Device (BYOD).
+ Online collaboration.
+ Video communications.
+ Cloud Computing.
+ Power-line Networking.
+ Wireless Broadband.


## Bring your own device:
+ BYOD allows end users to use personal tools to access the corporate network.
+ Allowing this trend can have major impacts on a network, such as security and compatibility with         corporate (شركة كبرى) software and devices.


## Cloud Computing:
+ Cloud computing is one of the ways that we access and store data.
+ There are four primary types of clouds: 
	+ Public clouds.
	+ Private clouds.
	+ Hybrid (هجين) clouds.
	+ Community (مجتمع) clouds.


## Powerline Networking:
+ Using a standard powerline adapter, devices can connect to the LAN wherever there is an electrical outlet (مَنفَذ).
+ No data cables need to be installed, and there is little to no additional electricity used.
+ Using the same wiring that delivers electricity, powerline networking sends information by sending data on certain frequencies.
