- Network Mapper (Nmap):
	It is still acclaimed today as one of the best tools for network  
	reconnaissance(الاستطلاع) and security auditing in cybersecurity.

-  Nmap tools:
	The Nmap project itself evolved into a family of advanced networking  
	tools that includes amazing projects such as Ncrack, Ncat, Nping, Zenmap, and, built into  
	Nmap itself, the Nmap Scripting Engine (NSE).


- download Namp (Precompiled packages):
	Precompiled Nmap packages can be found for all major platforms at 
	https://nmap.org/download.html for those who do not feel like setting up the build environment
