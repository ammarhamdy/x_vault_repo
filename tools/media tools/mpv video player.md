## To add your own script
```
$ locate mpv | grep script
```

if you install it with snap
```
/snap/mpv/201/usr/share/mpv/scripts
```
if you install it with apt
```
~/.config/mpv/scripts
```


## To start from specific time
```
mpv --start=3600 /path/to/video.mp4
```
