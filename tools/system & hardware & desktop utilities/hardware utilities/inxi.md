
# Commonly Used Options

|Command|Description|
|---|---|
|`inxi -S`|Show system info (OS, kernel, uptime, desktop).|
|`inxi -C`|Show CPU info.|
|`inxi -M`|Show motherboard details.|
|`inxi -G`|Show graphics (GPU + drivers + display).|
|`inxi -N`|Show network cards.|
|`inxi -D`|Show disks and storage devices.|
|`inxi -F`|Full system summary (basic info for all hardware).|
|`inxi -Fxz`|Full summary **without sensitive data** (good for sharing online).|
|`inxi -Fxxx`|Very detailed summary (maximum detail).|
