# Download Prebuilt Release

- Go to: [https://github.com/skylot/jadx/releases](https://github.com/skylot/jadx/releases)
- Download `jadx-x.x.x.zip` (latest release)
- Extract the ZIP
- Inside, you’ll find:
    - `jadx-gui` – GUI tool
    - `jadx` – Command-line tool


> ⚠️ **Requires Java 8 or higher** on your system.


```bash
cd /opt/jadx
./jadx -d output_java base.apk
```
