

The `id` command is used to print user and group information for a specified user.
```
id [options] username
```
When executed without an argument, the `id` command outputs information about the current user, allowing you to confirm your identity on the system.


## `id` command:
```
am@ubuntu:~$ id
uid=1000(am) gid=1000(am) groups=1000(am),4(adm),24(cdrom),27(sudo),30(dip),46(plugdev),100(users),118(lpadmin)
```
+ The output of the `id` command always lists the user account information first, using the user ID and username first: `uid=1000(am)`
+ After the username the primary group is listed, denoted by both the group ID and group name: `gid=1000(am) `
+ Other information listed includes the groups the user belongs to.


## `id root`:
```
am@ubuntu:~$ id root
uid=0(root) gid=0(root) groups=0(root)
```
