# Shortcuts

### **Command Navigation**
- **`Ctrl + A`**: Move the cursor to the beginning of the line.
- **`Ctrl + E`**: Move the cursor to the end of the line.
- **`Ctrl + U`**: Clear everything before the cursor.
- **`Ctrl + K`**: Clear everything after the cursor.
- **`Ctrl + W`**: Delete the word before the cursor.
- **`Ctrl + Y`**: Paste the last deleted text (yank).
- **`Alt + F`**: Move forward by one word.
- **`Alt + B`**: Move backward by one word.

### **Command Editing**
- **`Ctrl + L`**: Clear the screen (similar to `clear` command).
- **`Ctrl + T`**: Swap the last two characters before the cursor.
- **`Alt + T`**: Swap the last two words.
- **`Ctrl + _`**: Undo the last action.

### **Command History**
- **`Ctrl + R`**: Search the command history interactively.
- **`Ctrl + G`**: Exit the history search mode.
- **`Up Arrow`**: Recall the previous command from history.
- **`Down Arrow`**: Recall the next command from history.

### **Process Management**
- **`Ctrl + C`**: Kill the current process.
- **`Ctrl + Z`**: Suspend the current process.
- **`jobs`**: List suspended or background jobs.
- **`fg`**: Bring a suspended job to the foreground.
- **`bg`**: Resume a suspended job in the background.

### **File and Path**
- **`Tab`**: Auto-complete file and directory names.
- **`Ctrl + X + E`**: Open the current command in the default text editor.

### **Miscellaneous**
- **`!!`**: Re-execute the last command.
- **`!n`**: Re-execute the command at position `n` in the history.
- **`!<command>`**: Re-execute the last command starting with `<command>`.
- **`Ctrl + D`**: Log out of the shell or exit a terminal.
